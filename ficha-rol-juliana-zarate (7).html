Alertas QA — "La Calidosa"

Qué es: sistema de detección temprana. Su objetivo es anticipar cualquier anomalía en una tienda antes de que escale, no corregir algo que ya pasó.

Cómo se hace: un Apps Script lee la hoja QA-WH Tracker (y cinco pestañas exportadas de Aleph: Defectos, Stockouts, Cancelaciones, Pickers, Tiempos), compara cada tienda contra metas fijas por país y envía automáticamente un reporte estructurado los lunes 8am por Slack ("La Calidosa"), más alertas puntuales martes y jueves apenas una tienda se empieza a salir de meta.

Fuente de datos: QA-WH Tracker (Sheets) + exportes Aleph.

⚙️ 100% automatizado — no requiere ningún ajuste manual. También corre el reporte de visitas a tienda cada miércoles 8am (mismo bot).

function toMidnight(raw) {
  if (!raw) return null;
  const d = (raw instanceof Date) ? new Date(raw) : new Date(raw);
  if (isNaN(d.getTime())) return null;
  d.setHours(0, 0, 0, 0);
  return d;
}

function sameDay(d1, d2) {
  const a = toMidnight(d1), b = toMidnight(d2);
  if (!a || !b) return false;
  return a.getTime() === b.getTime();
}

function fmtDate(d) {
  if (!d) return "?";
  const dt = toMidnight(d);
  return `${String(dt.getDate()).padStart(2,'0')}/${String(dt.getMonth()+1).padStart(2,'0')}/${dt.getFullYear()}`;
}

function parseQ(val) {
  if (!val) return 99;
  return parseInt(val.toString().replace(/[^0-9]/g, '')) || 99;
}

function fmtScore(val) {
  if (!val && val !== 0) return "";
  const n = parseFloat(val);
  if (isNaN(n)) return "";
  return n.toFixed(1);
}

function mention(email) {
  if (!email) return "";
  const prefix = email.toString().split("@")[0].toLowerCase().trim();
  const id = SLACK_USERS[prefix];
  if (!id) return `@${prefix}`;
  if (id.startsWith("U")) return `<@${id}>`;
  return `@${id}`;
}

function criticalMetrics(row) {

function getTargetWeek() {
  const today = new Date();
  today.setHours(0, 0, 0, 0);
  const dow = today.getDay();
  const daysToThisMonday = dow === 0 ? 6 : dow - 1;
  const thisMonday = new Date(today);
  thisMonday.setDate(today.getDate() - daysToThisMonday);
  const target = new Date(thisMonday);
  target.setDate(thisMonday.getDate() - 7);
  return target;
}

function deltaStr(curr, prev) {
  const d = curr - prev;
  if (d > 0) return `▲+${d}`;
  if (d < 0) return `▼${d}`;
  return `→0`;
}

Ops review · Refuerzo semanal

WBR y planes de acción

Qué es: el review semanal de operación de Turbo (cancelaciones de partner, SO Ops, defect rate, compensaciones y disponibilidad, país por país), del que salen los planes de acción. A diferencia de las alertas, el plan de acción no anticipa: existe para reforzar y corregir puntualmente las métricas que ya cayeron la semana anterior.

Cómo se hace: se cargan los CSV de la semana → se analizan en Python/pandas → se genera un documento Word (.docx) con los hallazgos → se construye el plan de acción por país (tabla horizontal, listados de packing por tienda con EAN y acción por producto, historial de pickers reincidentes). Antes de armar el plan de la semana siguiente, siempre se valida primero si se cumplió el plan de la semana anterior cruzándolo contra la data nueva.

Flujo de publicación y seguimiento: con el documento ya armado, se pegan los planes de la semana en curso (formato estandarizado) en la pestaña "Planes" del sheet de TurboOps — no tiene trigger, así que se puede hacer a cualquier hora, preferiblemente en la mañana. Luego se corre el Apps Script SendPlanesDeAccion, que envía el plan a todos los grupos de Slack. El seguimiento (evidencia de los planes ejecutados) migra automáticamente a la pestaña "Form Responses 1" del mismo sheet. A fin de semana se valida qué se cumplió y qué no, se hace un push final por Slack, y a quienes cumplieron con evidencia se les suma +0.2pp al QA Score semanal — ese consolidado se lleva en la pestaña "QA Ajustado", acumulando puntos a favor para ajustar a fin de mes lo reportado en Instore Weekly FUP del tracker.

def normalize_name(s):
    """minúsculas + espacios colapsados + sin tildes, para que 'Efraín' y
    'Efrain', o 'Alberto  Del Moral' (doble espacio) y 'Alberto Del Moral'
    matcheen solos sin necesitar un alias manual para cada variante."""
    s = str(s).strip().lower()
    s = re.sub(r"\s+", " ", s)
    s = "".join(c for c in unicodedata.normalize("NFD", s) if unicodedata.category(c) != "Mn")
    return s
